# UI Kit React

React UI components library built with TailwindCSS.

## Installation

```bash
npm install /ui-kit-react-nguyenbao
```

## Usage

```jsx
import { Button } from "ui-kit-react-nguyenbao";

function App() {
  return <Button onClick={() => alert("Hello!")}>Click me</Button>;
}
```
